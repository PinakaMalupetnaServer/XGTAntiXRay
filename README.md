# XGTHelpOP

## Description


This is a PocketMine Plugin.

This plugin notifies the staff when someone breaks an ore

## How to install

1. Install plugin

2. Stop/start server

3. Edit config

4. Restart server

5. Have FUN!


## Features
- [X] Config

- [X] Costom Warnings(Place/Break)


## Permissions

Break

  xgtantixray.break

Place

  xgtantixray.place

## Config

```
# 0 = false
# 1 = true

# Event | Select when to send a warning
Break: 1 
Place: 0

# Ores | Select the type of ore to send a warning
IronOre: 0
CoalOre: 0
GoldOre: 0
RedstoneOre: 0
LapisOre: 1
EmeraldOre: 0
DiamondOre: 1
```
